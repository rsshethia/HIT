Printing Instructions

For best results when printing:

  • Use the browser's print function (Ctrl+P or Cmd+P)
  • Set paper size to A4 or Letter
  • Enable "Background graphics" to preserve colors
  • Set margins to "None" or minimal
  • Each card will print on a separate page

For a printable version that fits on credit card sized paper:

  • Set destination to "Save as PDF"
  • In More Settings, disable headers and footers
  • Set paper size to custom 86mm √ó 54mm if available
  • Print the PDF on card stock and cut along the edges

These instructions will not appear in the printed document.
